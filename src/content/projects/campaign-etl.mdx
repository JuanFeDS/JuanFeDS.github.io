## Contexto

Una empresa de retail necesitaba consolidar datos de campañas publicitarias dispersos en múltiples CSVs exportados manualmente desde distintas plataformas (Meta Ads, Google Ads, email). El proceso era manual, propenso a errores y tomaba medio día cada semana.

## Problema

- Datos en formatos distintos por plataforma
- Sin estandarización de métricas (cada plataforma llama distinto a "impresiones", "alcance", etc.)
- Sin historial centralizado para análisis longitudinal

## Solución

Script Python que:

1. Lee los CSVs desde un directorio local (o GCS en versión extendida)
2. Normaliza esquemas por plataforma con mapeos de columnas
3. Aplica validaciones básicas de calidad (nulos, rangos de fechas, duplicados)
4. Carga el resultado consolidado a una tabla en BigQuery

```python
# Ejemplo del paso de normalización
COLUMN_MAP = {
    "meta": {"reach": "impressions", "spend": "cost"},
    "google": {"Impressions": "impressions", "Cost": "cost"},
}

def normalize(df: pd.DataFrame, platform: str) -> pd.DataFrame:
    return df.rename(columns=COLUMN_MAP[platform])
```

## Resultados

- Proceso que tomaba ~4 horas reducido a 8 minutos
- Historial de 18 meses centralizado en BigQuery
- Base para dashboards en Looker Studio
