## Contexto

Proyecto personal para aprender React Native. Terminó siendo una app funcional con usuarios reales.

## Features

- Tracking diario con check-in de un toque
- Rachas y estadísticas de consistencia
- Recordatorios adaptativos según hora de mayor cumplimiento histórico
- Resumen semanal generado por GPT-4o con observaciones personalizadas

## Stack

Backend en Node.js con PostgreSQL. Notificaciones push via Expo. El resumen semanal se genera en un cron job nocturno y se entrega vía push al despertar.
